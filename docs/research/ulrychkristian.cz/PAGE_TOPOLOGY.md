# Page Topology

1. Header
   - Fixed overlay, full width, top aligned.
   - Interaction model: hover and mobile menu affordance.

2. Hero
   - Large editorial headline, hero portrait, metadata grid.
   - Interaction model: static.

3. Showcase Strip
   - Large media band using Veevoy and Eqvista imagery.
   - Interaction model: static media.

4. Selected Work
   - Two-column staggered project grid on desktop, single column on mobile.
   - Interaction model: link hover.

5. Services
   - Six-cell bordered service grid and contact CTA row.
   - Interaction model: CTA hover.

6. Testimonial
   - Large quotation with right-aligned attribution.
   - Interaction model: static.

7. Articles
   - Two article cards with real images and links.
   - Interaction model: link hover.

8. Contact
   - Large gradient email link.
   - Interaction model: mailto link hover.

9. Footer
   - Build credit, Prague time text, privacy link, social pills.
   - Interaction model: link hover.

10. Awwwards Badge
   - Fixed right edge badge.
   - Interaction model: external link.
