# Behaviors

## Global
- `html` includes `lenis custom-cursor-active` on the target.
- The target has a dark starfield-like animated background and a fixed Awwwards nominee badge on the right side.
- Clone uses a CSS starfield and fixed badge. Native smooth scrolling is enabled.

## Header
- Fixed at top with translucent dark background and backdrop blur.
- Desktop nav shows brand, role, Work, About, EN selector, and Get in touch button.
- Mobile target collapses nav to a small menu icon.
- Hover states: text links brighten; primary buttons invert from white fill to transparent dark.

## Hero
- Static editorial layout.
- Desktop: large Newsreader headline on the left, portrait placed lower on the right, metadata at bottom left.
- Mobile: headline, metadata, then portrait.
- Accent word "Prague" and email CTA use the extracted lavender gradient text.

## Media And Work
- Work/project media is scroll-revealed on the target.
- Clone renders all media directly with hover scale for cards.
- Project cards link to the same URL paths as the source.

## Services
- Static grid with six services, separated by thin borders.
- CTA row at the bottom links to contact.

## Testimonial, Articles, Footer
- Static sections.
- Article cards link to the same article paths as the source.
- Footer social links are external and open in new tabs.
